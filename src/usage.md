# Usage Guide

## Command Line Options

- `--mode` : Choose scanning algorithm (`kangaroo`, `bsgs`, `random`)
- `--pubkey` : Target public key (hex)
- `--range` : Range for kangaroo/bsgs (e.g., `0:2^60`)
- `--address` : Bitcoin address to check
- `--output` : File to save found keys
